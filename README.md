# PreventLockScreen  

PreventLockScreen 是一个极简的 Windows 工具，用于 **防止系统自动锁屏**。  
整个程序只有 **32KB 大小的 exe 文件**，无需安装，双击即可运行。

## ✨ 功能特点  
- **防止锁屏**：阻止 Windows 因长时间无操作而进入锁屏。  
- **超轻量**：仅 32KB 的可执行文件，几乎不占磁盘和内存。  
- **即开即用**：无需安装，双击运行即可生效。  
- **后台常驻**：可最小化到任务栏，持续保持屏幕常亮。  

---

## 🚀 使用方法  
1. 下载最新的 [Release](https://github.com/Char-CN/PreventLockScreen/releases)。  
2. 解压并运行 `PreventLockScreen.exe`（仅 32KB）。  
3. 程序启动后会自动阻止系统锁屏。  

---

## 🖥️ 适用场景  
- 演示或会议时保持屏幕常亮  
- 长时间观看视频或监控数据  
- 远程桌面或自动化任务执行过程中避免锁屏
