# PreventLockScreen  

PreventLockScreen 是一个轻量级的 Windows 桌面工具，用于 **防止系统自动锁屏**。当你需要长时间保持屏幕常亮（例如演示、会议、看视频或监控任务），这个工具可以帮你避免频繁输入密码或解锁。  

---

## ✨ 功能特点  
- **防止锁屏**：阻止 Windows 在空闲时自动进入锁屏。  
- **轻量运行**：程序体积小，运行时占用资源极低。  
- **简单易用**：启动后即可生效，无需复杂配置。  
- **图形界面**：提供基础的 Windows 窗体界面，方便操作。  

---

## 📂 项目结构  
- `Program.cs`：程序入口。  
- `Form1.cs`：主窗体逻辑。  
- `PreventLockScreen.csproj`：项目文件。  
- `App.config`：应用配置。  
- `app.ico`：应用图标。  

---

## 🚀 使用方法  
1. 下载最新的 [Release](https://github.com/Char-CN/PreventLockScreen/releases)。  
2. 解压并运行 `PreventLockScreen.exe`。  
3. 程序启动后会自动阻止系统锁屏，你可以最小化到任务栏继续使用。  

---

## 🖥️ 应用场景  
- 演示或会议时保持屏幕常亮。  
- 长时间观看视频或监控数据。  
- 远程桌面或自动化任务执行过程中避免锁屏。
